使用说明
==============================

### 代码功能

作为39F数据中心网站的上传功能母版，只需简单修改即可使用

### 使用环境

使用语言：php、python(anaconda)

适用范围：39F数据中心网站

### 部署步骤

1.在数据库中创建一个新表

2.将test.php按需求修改作为网站的新上传页面

3.将upload.php中内容修改后复制进原文件中

4.将BASE.py中内容修改后复制进原文件中

5.将upload_test.py按需求以及数据库数据类型修改后加入python_codes文件夹中
